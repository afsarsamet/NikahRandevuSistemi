# NikahRandevuSistemi
Bu proje, kullanıcıların randevu oluşturabildiği ve yönetebildiği bir masaüstü uygulamasıdır.

## 🚀 Özellikler
- Randevu oluşturma, listeleme ve silme (CRUD işlemleri)
- SQL veritabanı entegrasyonu
- Randevu çakışma kontrolü
- Kullanıcı kayıt ve giriş sistemi (şifreler hash’lenerek saklanır)
- n8n ile otomatik e-posta gönderimi
- Webhook ile uygulama ve otomasyon sistemi entegrasyonu

## 🛠️ Kullanılan Teknolojiler
- C# (Windows Forms)
- SQL Server
- n8n (Workflow Automation)
- REST API & Webhook

## ⚙️ Nasıl Çalışır?
1. Kullanıcı sisteme giriş yapar veya kayıt olur  
2. Randevu oluşturur  
3. Randevu bilgileri veritabanına kaydedilir  
4. n8n webhook tetiklenir  
5. Kullanıcıya otomatik e-posta gönderilir
